# Colevity Landing Page

## Tech choice
**Vite + React + TypeScript + Tailwind CSS** — fast local dev, easy static build, simple deployment to Vercel/Netlify, and minimal runtime overhead for a single-page site.

## Getting started
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Deploy
- **Vercel**: import the repo, set build command `npm run build`, output `dist/`.
- **Netlify**: build command `npm run build`, publish directory `dist/`.

## Where to edit copy
- `src/App.tsx` — all page content and sections.
- `index.html` — SEO/meta tags.
- `public/` — favicon + OG image placeholders.
